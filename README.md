## 🚀 Featured Project

### [🎬 Tube-Tweet](https://github.com/itskkkk/Tube-Tweet) · [Live Demo →](https://tube-tweet-kohl.vercel.app/)

> A full-stack video hosting + micro-blogging platform built with the MERN stack — deployed and production-ready.

**What it does:**
- 🎥 Video upload, playback, publish/unpublish control
- 🔐 JWT + bcrypt authentication with access & refresh tokens
- 📋 Playlists, subscriptions, watch history, liked videos
- 💬 Comments, tweets, likes/dislikes across content types
- 📊 Channel dashboard with real-time stats (views, subscribers, likes)
- 🔍 Search with pagination

**Tech:** React · Node.js · Express · MongoDB · Mongoose · JWT · Cloudinary · Tailwind CSS
